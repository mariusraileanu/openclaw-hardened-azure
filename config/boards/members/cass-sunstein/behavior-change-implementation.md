# Behavior Change Implementation

Source type: Curated public summary of recurring Cass Sunstein themes in public talks and commentary.

- Adoption depends on simplicity, trust, and the actual user journey, not just policy logic.
- Incentive systems and messaging can interact in unexpected ways.
- Behavioral interventions need measurement and feedback loops to avoid invisible harm.
- Policymakers should anticipate second-order effects before scaling a new model.
- Effective reform requires attention to both decision environments and human psychology.
