# Value-Based Care Principles

Source type: Curated public summary of recurring Mark McClellan themes in value-based care and payment reform.

- Payment reform works only when incentives line up with outcomes that providers can actually influence.
- Evaluation needs explicit metrics, risk adjustment, and accountability, not vague aspirations.
- Pilots should be scalable and financially sustainable if they succeed.
- Poor incentive design can create risk selection, gaming, or under-service.
- Reimbursement reform should reward better care, not just more reporting burden.
